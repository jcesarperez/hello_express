# hello_express
Wellcome to Hello express project

Just playing with node and express

A webapp with routes:
* /: a welcome message
* /users: a users listing

# requires
* node: https://nodejs.org/es/download/

# install
```
git clone https://github.com/jcesarperez/hello_express.git
cd hello_express
npm install
```

# run
```
npm start
```
Check http://localhost:3000/
Check http://localhost:3000/users
